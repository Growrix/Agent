## Composition Primitives by Route

### High-Latitude Routes (HOME, PORTFOLIO)
These routes have creative freedom within these primitives:
- **Reveal Strategy:** Staggered entrance animations (text, cards, images)
- **Surface Stack:** 2-3 layer composition (text overlay on image, cards above background)
- **Rhythm:** Alternating section heights (short hero, tall services, medium testimonials)
- **Asymmetry:** Left-aligned text sections alternate with right-aligned image sections

### Medium-Latitude Routes (SERVICES, FINANCING, ABOUT, BLOG)
These routes follow recommended patterns but allow deviation:
- **Reveal Strategy:** Smooth entrance animations (fade, slide)
- **Surface Stack:** 1-2 layer composition (cards on background, simple layouts)
- **Rhythm:** Consistent section heights (grid-based, predictable)
- **Asymmetry:** Symmetrical layouts allowed (centered grids, balanced columns)

### Low-Latitude Routes (CONTACT, FAQ)
These routes enforce strict structure:
- **Reveal Strategy:** Instant loading or simple fade (focus on functionality, not showiness)
- **Surface Stack:** Minimal layers (form on background, no overlay complexity)
- **Rhythm:** Predictable, linear progression
- **Asymmetry:** Centered, symmetrical layouts only
