# Component Spec: QuoteCalculatorPanel

- Class: molecule
- Variants: inline_page, modal_entry
- Content keys: quote.calculator.*, quote.form.*, quote.success.*, quote.error.*
- Data source: integration.quote.estimate(payload)
- Form contract: zod schema with preserved input on errors
- ARIA: labelled form fields, describedby helper and errors, aria-busy while submitting
- Responsive: multi-panel desktop, stacked step cards mobile
- Motion: motion.counter.quote-update and motion.input.focus-ring; reduced_motion direct updates
- States:
  - default
  - hover
  - focus-visible
  - active
  - disabled
  - loading (estimate and submit)
  - success (lead accepted)
  - validation_error
  - server_error
  - empty (insufficient input)
