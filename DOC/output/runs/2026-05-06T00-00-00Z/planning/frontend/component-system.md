## Required State Matrix Policy
Every interactive component explicitly declares:
- default
- hover
- focus-visible
- active
- disabled
- loading when applicable
- success, error, empty where applicable

## Accessibility and ARIA Baseline
- keyboard reachability and logical tab order
- visible focus treatment via focus tokens
- disclosure controls with aria-expanded and aria-controls
- modal controls with aria-modal, labelledby, describedby
- live region behavior for async feedback
