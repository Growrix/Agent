# Plumbing Service Website Frontend Plan

This folder is the planning set for the Australian plumbing service website on the `Plumber_test` branch.

## Purpose
- Build a modern, high-trust, mobile-first plumbing site for Australia.
- Keep call, WhatsApp, and AI chat assistance visible on every page.
- Support emergency repairs, planned bookings, postcode coverage checks, quote requests, and service-area SEO.
- Use the site as a conversion engine, not a brochure.

## AI First Stop
- Start with [ai-context.yaml](ai-context.yaml).
- Then read [00-master-ui-architecture.md](00-master-ui-architecture.md), [01-design-system.md](01-design-system.md), [02-component-system.md](02-component-system.md), and [03-page-plan.md](03-page-plan.md).

## Planning Scope
- Home and service discovery.
- Emergency triage and quote estimation.
- Postcode / area coverage.
- Booking and contact flows.
- Reviews and trust proof.
- AI concierge and WhatsApp escalation.
- Blog / resource SEO surface.
- Utility pages such as FAQ, privacy, terms, and 404.
