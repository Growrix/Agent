# MCP Servers Service Page

## Page Definition
- Purpose: explain Model Context Protocol server development clearly and convert technical buyers.
- Target audience: AI product teams, internal platform teams, and businesses building agent workflows.
- Primary CTA: Scope an MCP Server.
- Secondary CTA: Browse MCP Products.

## Sections In Visual Order

### 1. Hero With Plain-Language Framing
- Content: what MCP servers are, why they matter, and how they unlock agent capabilities securely.
- Components: hero, explainer badges, architecture preview panel.

### 2. Use Cases
- Content: internal tools, CRM access, document retrieval, workflow orchestration, ecommerce support, developer tooling.
- Components: feature blocks, icon cards.

### 3. Architecture Overview
- Content: client, tool layer, auth, data sources, observability, governance.
- Components: diagram block, content tiles, code preview card.

### 4. Service Scope
- Content: API wrapping, auth flows, schema design, tool exposure, deployment, monitoring, handoff.
- Components: detail cards, checklist grid.

### 5. Productized Options
- Content: starter MCP server templates, vertical accelerators, custom integrations, maintenance plans.
- Components: pricing tiers, product cards.

### 6. Proof and Trust
- Content: technical credibility, delivery process, security posture, relevant project teasers.
- Components: stat tiles, testimonial, process strip.

### 7. FAQ
- Content: security, hosting, auth, supported systems, maintenance, pricing.
- Components: accordion.

### 8. Final Conversion Section
- Content: scoping CTA, AI concierge for technical questions, WhatsApp for quick assessment.
- Components: CTA band, action bar.
