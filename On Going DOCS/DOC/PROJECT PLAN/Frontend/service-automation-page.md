# Automation Service Page

## Page Definition
- Purpose: sell automation as business leverage with measurable operational wins.
- Target audience: operations leaders, founders, and teams buried in repetitive workflows.
- Primary CTA: Audit My Workflow.
- Secondary CTA: View Automation Examples.

## Sections In Visual Order

### 1. Hero With Operations Positioning
- Content: headline around reducing manual work, increasing speed, and connecting systems.
- Components: hero, automation flow visual, CTA pair.

### 2. Pain Point Grid
- Content: lead routing, content workflows, CRM sync, support triage, reporting, onboarding, approvals.
- Components: issue cards, icon list.

### 3. Automation Opportunities
- Content: what gets automated, expected business effect, typical systems involved.
- Components: feature blocks, stat cards.

### 4. Workflow Design Method
- Content: mapping, exceptions, approvals, observability, human handoff.
- Components: step indicator, process cards.

### 5. Example Automations
- Content: three to five example flows with before and after metrics.
- Components: case cards, comparison rows.

### 6. Integration Stack
- Content: APIs, webhook systems, CRMs, CMS, chat platforms, payments, internal tools.
- Components: logo grid, content block.

### 7. Engagement Models and FAQ
- Content: audit sprint, implementation package, ongoing optimization, support.
- Components: pricing tiers, accordion.

### 8. Final CTA
- Content: book audit, chat about a workflow, WhatsApp escalation.
- Components: CTA section, action bar.

## State Requirements
- Example flows need collapsed and expanded detail states.
- Audit CTA preselects automation service in booking form.

## Responsive Adaptation
- Workflow diagrams simplify into vertical steps on mobile.
- Integration stack becomes horizontally scrollable chips.

## SEO and Metadata
- Title: Business Automation Services for Operations, Sales, and Support.
- Description: Design and build practical automation systems that remove repetitive work and improve operational speed.

## Conversion Path
- Hero -> pain points -> example automations -> audit CTA.
