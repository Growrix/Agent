# Websites Service Page

## Page Definition
- Purpose: convert businesses that need visually differentiated, high-conversion websites.
- Target audience: founders, brands, agencies, and operators who need a premium web presence.
- Primary CTA: Plan My Website.
- Secondary CTA: View Website Portfolio.

## Sections In Visual Order

### 1. Hero With Visual Proof
- Content: positioning around standout web experiences, conversion, speed, and visual polish.
- Components: hero, before-and-after visual strip, CTA pair.

### 2. Website Types
- Content: marketing sites, company sites, product launch sites, ecommerce fronts, landing pages, content-rich sites.
- Components: filter chips, feature cards.

### 3. Frontend Differentiation
- Content: design system quality, motion discipline, responsive behavior, mobile app-like interactions where useful.
- Components: comparison panels, token callouts, motion notes.

### 4. Conversion Architecture
- Content: CTA strategy, trust sections, proof sequencing, lead capture, booking, chat, WhatsApp.
- Components: process cards, conversion map graphic.

### 5. Performance and SEO Section
- Content: Core Web Vitals, semantic structure, image handling, metadata, accessibility.
- Components: metric tiles, checklist blocks.

### 6. Featured Website Projects
- Content: selected projects with visual previews, performance wins, and conversion outcomes.
- Components: portfolio grid, quote strip.

### 7. Website Packages and Process
- Content: launch sprint, redesign track, conversion optimization retainer.
- Components: pricing tiers, step indicator.

### 8. CTA and FAQs
- Content: objections around timelines, content support, CMS handoff, revisions.
- Components: accordion, CTA band.

## State Requirements
- Portfolio previews support hover and touch-friendly expand behavior.
- Package comparison supports selected and recommended states.

## Responsive Adaptation
- Mobile emphasizes visual previews in vertical device frames.
- Performance section becomes concise proof cards instead of a broad metrics table.

## SEO and Metadata
- Title: Premium Website Design and Development for Modern Brands.
- Description: Design-forward websites with strong conversion architecture, speed, SEO readiness, and premium mobile experiences.

## Conversion Path
- Hero -> frontend differentiation -> project proof -> packages -> booking.
