# Design Modes

Mode definitions for the Design Decision System.

## What Is A Mode?

A mode controls **composition behavior** — how sections are structured, how rhythm flows, and how much visual tension the page carries. It does not control color, typography weight, or surface treatment — that is the preset's job.

## Available Modes

| Mode | File | Best For |
|------|------|----------|
| Clean SaaS | `CLEAN-SAAS.md` | Dashboard, settings, admin, utility, docs |
| Bold Marketing | `BOLD-MARKETING.md` | Homepage, launch, high-conversion landing |
| Editorial Narrative | `EDITORIAL-NARRATIVE.md` | Blog, case study, about, story-led content |
| Dense Data | `DENSE-DATA.md` | Analytics, reports, comparison, pricing |
| Mobile Native | `MOBILE-NATIVE.md` | App-like, mobile-first flows |

## How To Choose

Use `../OPERATING/MODE-SELECTION.md` for the selection algorithm and quick matrix.

## Loading Rule

Load only the mode file that matches the page decision. Do not load all modes at once.
