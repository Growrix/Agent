# Preset: Neutral Professional

Clean, reliable, corporate-friendly aesthetic. The safe default that works for most B2B and enterprise contexts.

## Character

- trustworthy
- clean
- predictable

## When To Use

- Enterprise B2B products
- Professional services platforms
- Default preset when no creative brief exists
- Internal tools and admin interfaces
- Contexts where "looking professional" matters more than standing out

## Visual Rules

### Typography

- Posture: product-neutral
- Headlines: medium weight, clear hierarchy, no dramatic size jumps
- Body: standard readable sizes, generous line height
- Display text: restrained — no oversized statements

### Color

- Accent: corporate blue, green, or neutral accent — low saturation preferred
- Backgrounds: white, light gray, or very subtle warm/cool tints
- Contrast: standard WCAG compliance, no extreme high-contrast drama
- Gradients: minimal or none

### Shape

- Language: balanced
- Radius: medium (8-12px range)
- Cards: standard with subtle border or light shadow
- Buttons: standard radius, clear states

### Surface

- Depth: restrained-depth
- Shadows: subtle, functional (elevation hierarchy only)
- Borders: light, consistent
- Glass effects: none

### Motion

- Character: restrained
- Transitions: quick and functional (150-250ms)
- No decorative animations
- Hover states are standard — no dramatic transforms

### Imagery

- Mode: product UI screenshots or clean illustrations
- No abstract art, bold gradients, or editorial photography
- Icons: standard outline or filled, no custom illustration style

## DS Token Mapping

- Theme: `light` for public marketing websites unless the brief explicitly calls for dark
- `data-visual`: default or `sleek`
- `data-density`: `balanced` for product UI, `spacious` for public marketing websites
- Shape tokens: default DS values
- Shadow tokens: default DS values
- Motion tokens: default DS values

## What This Preset Avoids

- Strong personality that could polarize
- Bold color choices
- Dramatic typography
- Decorative elements without function
- Anything that feels like a startup or a magazine
