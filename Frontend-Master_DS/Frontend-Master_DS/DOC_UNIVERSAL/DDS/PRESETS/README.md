# Preset Families

Visual preset definitions for the Design Decision System.

## What Is A Preset?

A preset controls **aesthetic personality** — how the page feels through color character, typography weight, shape language, surface treatment, and motion tone. It does not control layout structure or section composition — that is the mode's job.

## Available Presets

| Preset | File | Character |
|--------|------|-----------|
| Neutral Professional | `NEUTRAL-PROFESSIONAL.md` | Clean, safe, corporate-friendly |
| Editorial Premium | `EDITORIAL-PREMIUM.md` | Spacious, typographic, magazine-inspired |
| Warm Human | `WARM-HUMAN.md` | Soft, approachable, rounded, friendly |
| Technical Sharp | `TECHNICAL-SHARP.md` | Precise, high-contrast, developer-oriented |
| High-Energy Launch | `HIGH-ENERGY-LAUNCH.md` | Bold, vibrant, high-contrast, startup-launch |

## How To Choose

Match the preset to the brand personality defined in the visual direction brief:

- Professional B2B → Neutral Professional
- Content-led or editorial brand → Editorial Premium
- Consumer-facing or community-focused → Warm Human
- Developer tools or technical products → Technical Sharp
- Startup launch or campaign → High-Energy Launch

## Mode × Preset Combinations

Any mode can pair with any preset. Some combinations are more natural:

| Mode | Natural Presets | Unusual But Valid |
|------|----------------|-------------------|
| Clean SaaS | Neutral Professional, Technical Sharp | Warm Human |
| Bold Marketing | High-Energy Launch, Editorial Premium | Technical Sharp |
| Editorial Narrative | Editorial Premium, Warm Human | Neutral Professional |
| Dense Data | Technical Sharp, Neutral Professional | — |
| Mobile Native | Warm Human, Neutral Professional | High-Energy Launch |

## Loading Rule

Load only the preset file that matches the page decision. Do not load all presets at once.
