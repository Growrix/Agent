# Hero Patterns

Reference catalog of hero composition patterns. Use these as inspiration after selecting a mode and preset.

## Statement + Product Proof

- Large headline with a clear value proposition
- Product screenshot, demo, or interactive preview as the visual anchor
- Primary CTA below or beside the statement
- Trust signals (logos, metrics) below the hero

**Best for**: SaaS homepage, feature landing, product launch

**Mode affinity**: Bold Marketing, Clean SaaS

## Split Hero

- Content on one side, visual on the other (typically 50/50 or 60/40)
- Asymmetric tension between text and image
- CTA aligned with the content side
- Visual anchor is a product screenshot, illustration, or diagram

**Best for**: Feature pages, comparison pages, B2B landing

**Mode affinity**: Bold Marketing, Editorial Narrative

## Editorial Collage

- Multiple visual elements arranged in an editorial layout
- Typography is the primary anchor — large, expressive headline
- Supporting elements (photos, shapes, quotes) create visual depth
- Less structured than a grid, more curated

**Best for**: About pages, brand stories, creative campaigns

**Mode affinity**: Editorial Narrative

## Metric-Led Hero

- One or more large numbers or statistics as the primary visual anchor
- Supporting context below or beside the metrics
- CTA tied to "get these results" messaging
- Works well with social proof

**Best for**: Case studies, results pages, pricing justification

**Mode affinity**: Bold Marketing, Dense Data

## Story-Led Hero

- Opening narrative or question that draws the reader in
- Minimal chrome — typography carries the hero
- Supporting image or subtle background treatment
- Scrolling reveal for the next section

**Best for**: Blog articles, case studies, about pages, editorial content

**Mode affinity**: Editorial Narrative

## Feature Stack Hero

- Multiple feature highlights stacked or arranged in a visual grid
- Each feature has an icon, title, and brief description
- No single dominant visual anchor — the collection is the anchor
- Works for products with many capabilities

**Best for**: Feature overview, product tour, docs landing

**Mode affinity**: Clean SaaS, Bold Marketing

## Media-Led Hero

- Full-width video, animation, or interactive element dominates the viewport
- Minimal text overlay — the media tells the story
- CTA appears after or alongside the media
- Requires high-quality media assets

**Best for**: Product demos, creative campaigns, event pages

**Mode affinity**: Bold Marketing

## Context Bar Hero (Utility)

- No traditional hero — replaced by search, filters, or navigation
- Page title and breadcrumbs provide context
- Content starts immediately after the bar
- Functional, not decorative

**Best for**: Dashboard, admin, docs, search results, data pages

**Mode affinity**: Clean SaaS, Dense Data
