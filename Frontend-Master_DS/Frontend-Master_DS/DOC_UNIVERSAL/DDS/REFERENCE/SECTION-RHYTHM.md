# Section Rhythm

Reference patterns for pacing and rhythm within page composition.

## What Is Section Rhythm?

Section rhythm is the pattern of visual weight, density, and contrast as the user scrolls through a page. Good rhythm keeps the user engaged. Bad rhythm makes the page feel monotonous or exhausting.

## Rhythm Patterns

### The Wave

```
[HIGH] Hero — bold, dominant
[LOW]  Breathing section — simple statement or transition
[MED]  Feature section — balanced detail
[LOW]  Quote or proof — editorial pause
[HIGH] Conversion section — strong CTA
[LOW]  Footer or closing
```

Best for: Marketing pages, landing pages.

### The Escalator

```
[LOW]  Soft intro — context setting
[MED]  Feature explanation — building understanding
[MED+] Proof section — building trust
[HIGH] Comparison or deep feature — increasing commitment
[MAX]  Conversion section — peak energy
```

Best for: Long-form landing pages, pricing pages.

### The Steady Beat

```
[MED] Section 1
[MED] Section 2
[MED] Section 3
[MED] Section 4
```

Best for: Dashboard, settings, admin pages. Not recommended for marketing.

### The Reset

```
[HIGH] Hero
[MED]  Features
[MED]  More features
[LOW]  Full-width reset — different background, editorial moment
[MED]  Proof section
[HIGH] Final CTA
```

Best for: Homepage, about page — breaks monotony in the middle.

### The Funnel

```
[WIDE]    Full-width hero
[MEDIUM]  Constrained feature section
[NARROW]  Focused reading section
[FOCUSED] Single CTA block
```

Best for: Conversion-focused landing pages.

## Contrast Techniques

Ways to create visual contrast between sections:

- **Background shift**: alternating between light and dark surfaces
- **Width change**: full-width section followed by constrained-width
- **Density shift**: spacious section followed by compact data
- **Content type change**: text-heavy followed by image-heavy
- **Layout change**: grid followed by single-column
- **Scale change**: large type section followed by small-text detail

## Anti-Patterns

- All sections same background, same width, same visual weight
- No pause or reset in a long page
- Escalation without a payoff (building energy that goes nowhere)
- Only using color to create contrast (ignoring layout and scale)
- Three or more consecutive card-grid sections
