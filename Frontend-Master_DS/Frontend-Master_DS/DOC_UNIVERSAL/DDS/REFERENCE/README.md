# DDS Reference

Non-authoritative inspiration and pattern reference for creative frontend work.

## Purpose

These files provide examples and patterns to inspire composition decisions. They are not rules — the OPERATING files are the rules.

## Available References

| File | Content |
|------|---------|
| `HERO-PATTERNS.md` | Hero composition patterns with use cases |
| `SECTION-RHYTHM.md` | Section rhythm and pacing patterns |
| `LAYOUT-COMPOSITION-PATTERNS.md` | Layout structures and composition techniques |

## Loading Rule

Load reference files only when:

- Seeking inspiration for a specific composition problem
- The mode and preset are already selected
- The wireframe brief needs pattern examples

Do not load reference files as part of the standard AI execution flow.
