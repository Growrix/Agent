# Next.js SaaS Profile

On-demand technical handbook for a Next.js SaaS stack.

## Purpose

This profile is not always-read. It exists so AI and developers can load only the stack-specific guidance needed for the current task.

## Load This Profile When

- building or refactoring Next.js App Router flows
- shaping backend contracts in a Node-based SaaS app
- reviewing design system usage, security, testing, deployment, or operations for this stack

## Suggested Profile Subsets

- frontend task: `coding-standards/`, `theming/`, `architecture/`
- backend task: `coding-standards/`, `architecture/`, `security/`
- release task: `devops/`, `operations/`
- public performance task: `seo-performance/`

## Included In This Curated Profile

- reusable architecture docs
- coding standards for React, Next.js, Node, TypeScript, git, and folder discipline
- theming and UI guidance
- security handbook files
- testing handbook files
- devops, operations, and SEO-performance docs

## Intentionally Not Copied As-Is

- duplicate constitution-level authority docs
- project product vision or project overview docs
- feature-specific notification planning docs
- obviously product-specific reference files

Those files may still exist elsewhere in the repo, but they are not part of this reusable stack profile.

## Curation Rule

Only move reusable, generic, stack-level docs into this profile. Do not copy project-specific product rules into it.
