# Glossary

- `AI runner`: the execution agent, such as Copilot or Aider.
- `API map`: the contract index for endpoints, server actions, and mutations.
- `dashboard shell`: the shared authenticated layout containing navigation and framing.
- `design system`: the token-driven visual system used by pages and features.
- `done criteria`: the observable conditions that must be true before a task is complete.
- `quality gate`: a required validation step such as build, lint, typecheck, or tests.
- `route map`: the source of truth for routes, layouts, access rules, and nav placement.
- `stack profile`: on-demand handbook for a specific technology stack.
- `task queue`: the active execution list that drives implementation.
