# FRONTEND Folder Documentation

এই ফোল্ডারটি ফ্রন্টএন্ড ডিজাইন, কম্পোনেন্ট, এবং নির্দেশনা সংক্রান্ত সকল গুরুত্বপূর্ণ ফাইল সংরক্ষণ করে। প্রতিটি ফাইলের উদ্দেশ্য এবং ব্যবহার নিচে বাংলা ও ইংরেজিতে ব্যাখ্যা করা হয়েছে।

---

## 1. Design-System.md
- **English:**
  The single source of truth for all design tokens, component styles, usage guidelines, and mobile app-like design patterns. Use this file for every visual and brand decision in the frontend.
- **বাংলা:**
  ফ্রন্টএন্ডের সকল ডিজাইন টোকেন, কম্পোনেন্ট স্টাইল, ব্যবহারের নির্দেশনা এবং মোবাইল অ্যাপ-স্টাইল ডিজাইনের জন্য প্রধান রেফারেন্স। সকল ভিজ্যুয়াল ও ব্র্যান্ড সিদ্ধান্তের জন্য এই ফাইলটি ব্যবহার করুন।

## 2. Component-Library-Checklist.md
- **English:**
  A comprehensive checklist of all required UI components, including primitives, layouts, navigation, feedback, forms, media, admin, public, and utility components. Use this to track and plan component development.
- **বাংলা:**
  সকল প্রয়োজনীয় UI কম্পোনেন্টের চেকলিস্ট, যেমন প্রিমিটিভ, লেআউট, ন্যাভিগেশন, ফিডব্যাক, ফর্ম, মিডিয়া, অ্যাডমিন, পাবলিক ও ইউটিলিটি কম্পোনেন্ট। কম্পোনেন্ট ডেভেলপমেন্ট পরিকল্পনা ও ট্র্যাক করার জন্য ব্যবহার করুন।

## 3. Frontend-System-Audit-Instruction-2026.md
- **English:**
  Contains audit instructions and standards for reviewing and improving the frontend system. Use this file to ensure quality, consistency, and best practices in frontend development.
- **বাংলা:**
  ফ্রন্টএন্ড সিস্টেম পর্যালোচনা ও উন্নয়নের জন্য অডিট নির্দেশনা ও স্ট্যান্ডার্ড। ফ্রন্টএন্ড ডেভেলপমেন্টে কোয়ালিটি, কনসিস্টেন্সি ও বেস্ট প্র্যাকটিস নিশ্চিত করতে এই ফাইলটি ব্যবহার করুন।

## 4. Frontend-Design-System.md
- **English:**
  An advanced, detailed design system file with explicit rules for color usage, typography, spacing, motion, dark mode, z-index, iconography, image/media, forms, i18n, documentation, and testing. Reference for all design and implementation work.
- **বাংলা:**
  কালার, টাইপোগ্রাফি, স্পেসিং, মোশন, ডার্ক মোড, জেড-ইন্ডেক্স, আইকনোগ্রাফি, ইমেজ/মিডিয়া, ফর্ম, i18n, ডকুমেন্টেশন ও টেস্টিং-এর জন্য বিস্তারিত নিয়মাবলী সহ উন্নত ডিজাইন সিস্টেম। সকল ডিজাইন ও ইমপ্লিমেন্টেশনের জন্য রেফারেন্স।
