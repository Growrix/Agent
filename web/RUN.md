# Run Guide

## Prerequisites
- Node.js LTS
- npm

## Install
- npm install

## Development
- npm run dev

## Build
- npm run build

## Lint
- npm run lint

## Unit Tests
- npm run test

## E2E Tests
- npm run test:e2e

## Smoke Checklist
- Home route loads with hero, grid, testimonials, and quote panel.
- Services and portfolio dynamic routes resolve and handle not-found.
- Quote page supports calculator interactions.
- Support action cluster opens call, WhatsApp, and assistant modal.
- Auth routes render sign-in and sign-up forms.
