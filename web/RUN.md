# Run Guide

## Prerequisites
- Node.js 20+
- npm 10+

## Install
npm install

## Development
npm run dev

## Build
npm run build

## Lint
npm run lint

## Unit Tests
npm run test

## E2E (scaffold)
npm run test:e2e

## Smoke Checklist
- Home hero renders with CTA and trust chips
- Dark theme toggle switches html data-theme
- Mobile bottom nav appears under lg breakpoint
- Contact form submits to success state
- Storm route shows emergency contact path
